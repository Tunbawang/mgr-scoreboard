# Martial Grappling Rules - スコアボードシステム

## ファイル構成
- operator.html … スコアラー操作画面
- display.html  … 観客表示画面
- roster.html   … 選手名簿（Excelから貼り付け→試合番号クリックで反映）

## 使い方
1. この3ファイルを同じフォルダに置く
2. operator.html をダブルクリックして開く（メインの操作画面）
3. 上部の「選手名簿を開く」「観客表示を新しいタブで開く」からそれぞれ開く

## 複数コート・複数試合を同時運営する場合
この3ファイルのセットをコートの数だけコピーしてください。
各PCが完全に独立して動作するので、PC同士の通信は不要です。
roster.html には、そのPC（コート）が担当する対戦カードだけを貼り付けて使ってください。

例：
- コート1用フォルダ … operator.html / display.html / roster.html（コート1の対戦カードのみ）
- コート2用フォルダ … 同じ3ファイル（コート2の対戦カードのみ）

## 動作環境
- インターネット接続不要（オフラインで完全動作）
- Chrome / Edge / Firefox など主要ブラウザで動作確認
- データはブラウザの localStorage に保存されるため、同じPC・同じブラウザで operator.html と display.html を開く必要があります
